项目背景
```
普通后端会点js，不喜欢心智负担很重的前端框架，从Svelte和Solid简单纠结之后半随机选了Svelte。准备vite+svelte以及了解到Svelte Kit做底子做个自己平
时用的前端壳子，以后能更多的将注意力集中到事情的内容上，而不是疲惫于各种不可控且乱而重的东西上。
```
目标
```
开发一个服务于个人全栈开发者的前端的壳子，用来承载一些简单而小巧的前端应用。
```
